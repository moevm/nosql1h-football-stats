# nosql1h-football-stats

# Инструкция по локальному запуску
* python -m venv venv
* virtualenv venv
* venv\Scripts\activate
* pip install flask
* pip install pymongo
* pip install numpy
* python
* import flask
* exit()
* set FLASK_APP=nosql.py
* flask run

* Перейти по адресу http://localhost:5000/
